# TripAdvisorThis is a repo for managing machine readable artifacts for the [TripAdvisor APIs](https://www.tripadvisor.com/). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [TripAdvisor APIs](https://www.tripadvisor.com/).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](https://www.tripadvisor.com/)- [Developer](https://www.tripadvisor.com/)- [Twitter](https://twitter.com/TripAdvisor)- [GitHub](https://github.com/tripadvisor)The [APIs.json](https://github.com/api-evangelist/tripadvisor/blob/master/apis.json) index provides links to all the relevant links for the TripAdvisor APIs, including the OpenAPI and Postman if they are present.
